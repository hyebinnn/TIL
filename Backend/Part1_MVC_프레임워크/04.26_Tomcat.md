## Tomcat 서버
- Servlet Interface (Spec) 구현체
  - `Servlet`이란, 자바를 사용하여 웹을 만들 때 필요한 기술 → **클라이언트의 요청을 처리하고 결과를 반환**하는 자바 웹 프로그래밍 기술
